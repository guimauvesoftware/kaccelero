# routers

Routers of kaccelero.

## Installation

Add dependency to your `build.gradle(.kts)` or `pom.xml`:

```kotlin
api("dev.kaccelero:routers:0.1.0")
```

```xml

<dependency>
    <groupId>dev.kaccelero</groupId>
    <artifactId>routers-jvm</artifactId>
    <version>0.1.0</version>
</dependency>
```
